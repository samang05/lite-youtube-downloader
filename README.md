# Lite YouTube Downloader

## 소개
이 프로젝트는 복잡한 설정이나 무거운 외부 프로그램(FFmpeg 등) 설치 없이, 파이썬(Python)만으로 유튜브 영상과 오디오를 빠르고 간편하게 다운로드할 수 있는 도구입니다. 

## 주요 기능
- **간편한 미디어 다운로드:** 유튜브 링크(URL) 복사 및 붙여넣기만으로 즉시 다운로드 가능
- **맞춤형 포맷 지원:** 비디오와 오디오가 통합된 고화질 동영상(MP4) 및 원본 고음질 오디오 안전 추출
- **자동 파일 관리:** 다운로드 완료 시 자동으로 `downloads` 폴더를 생성하여 파일 정리
- **높은 호환성:** 외부 의존성을 최소화하고 재생목록 전체 다운로드 방지 등 안전한 예외 처리 적용

## 사용 방법
1. **다운로드 및 준비**
   - 우측 상단의 `[<> Code]` 버튼을 눌러 `Download ZIP`으로 코드를 다운로드하고 압축을 풉니다.
   - 터미널(또는 명령 프롬프트)을 열고 해당 폴더로 이동한 뒤, 아래 명령어를 입력하여 필수 라이브러리를 설치합니다.
     ```bash
     pip install -r requirements.txt
     ```
2. **실행**
   - 아래 명령어를 입력하여 프로그램을 실행합니다.
     ```bash
     python main.py
     ```
   - 터미널 안내에 따라 다운로드하고 싶은 유튜브 주소를 입력하고, 원하는 작업(1번: 영상, 2번: 오디오)을 선택합니다.
