# 🎥 YouTube Media Downloader (유튜브 미디어 다운로더)

복잡한 설정이나 무거운 프로그램 설치 없이, 실생활에서 빠르고 가볍게 사용할 수 있는 파이썬 기반 유튜브 다운로더입니다. 

## ✨ 주요 기능
- **간편한 다운로드**: 유튜브 URL 복사/붙여넣기만으로 즉시 다운로드 가능
- **동영상 다운로드 (MP4)**: 비디오와 오디오가 통합된 고화질/안전 포맷 자동 선택
- **오디오 추출**: 추가 인코딩 과정(FFmpeg 등) 없이 원본 고음질 오디오 스트림 안전 추출
- **자동 폴더 생성**: 다운로드 완료 시 자동으로 `downloads` 폴더를 생성하여 깔끔하게 파일 관리

## 🚀 시작하기

### 1. 환경 준비
이 프로그램은 외부 프로그램(FFmpeg) 설치 없이 파이썬만 있으면 작동합니다.
아래 명령어를 통해 필수 라이브러리인 `yt-dlp`를 설치해 주세요.

```bash
pip install -r requirements.txt
